Pre-Flight Checklist

  • Frontend server running on http://localhost:5173
  • Backend server running on http://localhost:5000
  • No errors in browser console (F12)
  • Browser cache cleared (Ctrl+Shift+Delete)
  • Page hard refreshed (Ctrl+F5)

Common Issues & Solutions

Issue 1: Button doesn't respond

Solution: Check browser console for JavaScript errors

Issue 2: Page doesn't load

Solution: Verify frontend server is running

Issue 3: 404 Not Found

Solution: Check route configuration in App.tsx

Issue 4: Blank page

Solution: Check Register component for errors
