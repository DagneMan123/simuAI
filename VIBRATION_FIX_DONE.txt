THE PROBLEM:
────────────────────────────────────────────────────────────
❌ Homepage was vibrating/shaking in browser
❌ Page was flickering and unstable
❌ Caused by infinite redirect loop


THE CAUSE:
────────────────────────────────────────────────────────────
Redirect components had wrong dependency arrays:
- useEffect([navigate]) ← navigate changes every render
- This caused infinite loop
- Loop = continuous redirects = vibration


THE FIX:
────────────────────────────────────────────────────────────
Changed to empty dependency arrays:
- useEffect([]) ← runs only ONCE
- No more infinite loop
- No more vibration!
