If Home Page is Blank or Hidden

Solution 1: Clear Browser Cache

  1. Press Ctrl + Shift + Delete
  2. Select "Cached images and files"
  3. Click "Clear data"
  4. Press Ctrl + F5 to hard refresh

Solution 2: Check Console for Errors

  1. Press F12 to open DevTools
  2. Go to "Console" tab
  3. Look for red errors
  4. If you see errors, share them

Solution 3: Try Incognito Mode

  1. Press Ctrl + Shift + N
  2. Go to http://localhost:5173
  3. If it works here, it's a cache issue

Solution 4: Check Servers Running

Common Issues

Issue: Page is completely blank
Cause: JavaScript error or cache issue
Fix: Clear cache + Check console for errors
Issue: Page shows "Cannot GET /"
Cause: Frontend server not running
Fix: Run cd frontend && npm run dev
Issue: Page loads but no content
Cause: Component rendering error
Fix: Check browser console (F12) for errors
